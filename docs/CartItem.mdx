# CartItem 컴포넌트
<Subtitle>장바구니의 개별 상품 정보 + 수량 조절/삭제</Subtitle>

<Description>
`CartItem`은 이미지, 브랜드, 가격, 수량을 표시하고 `onInc`, `onDec`, `onRemove` 콜백으로 상호작용을 제공합니다.

**Props 요약**
- `image`: 상품 이미지 (URL 또는 import된 자원)
- `brand`: 브랜드명 문자열
- `price`: 숫자(원 단위 표기; 내부에서 `toLocaleString()` 사용)
- `qty`: 현재 수량
- `onInc`/`onDec`/`onRemove`: 수량 증가/감소/삭제 콜백
</Description>

## 사용 방법
```jsx
import CartItem from './CartItem';
import shoes1 from '../assets/shoes_01.png';

<CartItem
  image={shoes1}
  brand="브랜드A"
  price={35000}
  qty={1}
  onInc={() => console.log('증가')}
  onDec={() => console.log('감소')}
  onRemove={() => console.log('삭제')}
/>;
